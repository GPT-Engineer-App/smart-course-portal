# smart-course-portal

create a learning management portal that is designed for college courses based on taking the course per semester. The instructor login should be able to manage the courses, and semester the course is available. instructor admin has access to students and student grades. student portal should display course information and progress of course taken.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/smart-course-portal.git
cd smart-course-portal
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
